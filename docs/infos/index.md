# 简介

在开发前可以通过此文档快速了解项目架构，此项目核心使用技术：`Vue3.4 + VueRouter + Pinia + elementPlus + TypeScript + Vite + Axios + Sass`；

## 前置环境

::: tip Node 版本
node>=18
pnpm>9
:::

## 一. 安装依赖

```
pnpm i
```

## 二. 项目启动

```
pnpm run dev
```

## 三. 项目打包

生产包

```
pnpm run build
```

测试包

```
pnpm run build:test
```

分析依赖包

```
pnpm run analyzer
```

## 开发文档docs运行

```
pnpm run docs:dev
```
