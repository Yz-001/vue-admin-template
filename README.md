# vue-admin-template

基于 `Vue3.4 + VueRouter + Pinia + elementPlus + TypeScript + Vite + Axios + Sass` 的后台管理系统

## 获取项目代码

```
git clone git@github.com:Yz-001/vue-admin-template.git
```

## 前置环境

::: tip Node 版本
node>=18
pnpm>9
:::

## 安装依赖

```
pnpm i
```

## 项目启动

```
pnpm run dev
```

## 项目打包

生产包

```
pnpm run build
```

测试包

```
pnpm run build:test
```

分析依赖包

```
pnpm run analyzer
```

## 开发文档docs运行

```
pnpm run docs:dev
```

更多信息可查看开发文档
