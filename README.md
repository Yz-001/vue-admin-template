# Vue 3 + TypeScript + Vite

基于 `Vue3.4 + VueRouter + Pinia + elementPlus + TypeScript + Vite + Axios + Sass` 的后台管理系统

## 前置环境

::: tip Node 版本
node>=18
pnpm>9
:::

## 一. 安装依赖

```
pnpm i
```

## 二. 项目启动

```
pnpm run dev
```

## 三. 项目打包

生产包

```
pnpm run build
```

测试包

```
pnpm run build:test
```

分析依赖包

```
pnpm run analyzer
```

## 开发文档docs运行

```
pnpm run docs:dev
```

更多信息可查看开发文档
